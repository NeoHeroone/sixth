Welcome to the Positioning Task Page

This page demonstrates various position properties in CSS. Scroll to see how each element behaves.

Static Position

This box uses static positioning, which is the default position for all elements. It stays in the normal flow of the document.

Relative Position

This box uses relative positioning. It is moved from its normal position using top and left properties.

Absolute Position

This box uses absolute positioning, placed inside a relatively positioned container to stay aligned.

Sticky Position

This navigation bar stays at the top of the viewport when you scroll, thanks to sticky positioning.
